## Decisions made while coding

- **Strategy shift mid-phase.** Plan said "P0 ends red, P1 rebuilds." Four of nine deletions couldn't land without their P1 replacements (no call site to point at, binary stopped building). Pair each structural deletion with its replacement in P1.0; the orthogonal four ship cleanly here.
- **Notification selection state stripped entirely.** Selection painting was the only consumer of `anchor` / `cursor_col`; whole `handle_mouse` impl + state removed. Toast selection rebuilds in P1 via the general path.
- **Lua `kind = "list"` panel removed.** `/agents`, `/resume` lose their list panels; P4 rebuilds list-of-buffers as N Windows in an Overlay.
- **Dead `BufferList` tests deleted, not migrated.** Six dialog.rs tests exercised general dialog behaviour using `BufferList`; coverage rebuilds against Overlay+Windows in P1.c.
